# Ladbrokes Landing Page

A modern, responsive landing page for Ladbrokes — the UK's most trusted sports betting brand.

## Features

- **Netflix-style hero** — perspective-tilted, scrolling grid of casino game tiles behind the headline
- **Scroll-driven animations** — sections scale up and take over the viewport as you scroll
- **Visual benefit cards** — rich CSS backgrounds with geometric shapes
- **Fully self-contained** — all assets embedded as base64; zero external image requests
- **Responsive** — optimised for desktop, tablet, and mobile

## Sections

1. **Hero** — Netflix-style casino tile background with headline offer
2. **Welcome Bonus** — step-by-step claim flow (Bet £5, Get £30)
3. **Why Ladbrokes** — six visual feature cards
4. **Partnerships** — Liverpool FC official betting partner
5. **Trust & Safety** — UKGC, SSL, GamStop, IBAS
6. **Final CTA** — conversion section with responsible gambling messaging

## Deployment

### GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages → Deploy from branch → main → / (root)**
3. Live at `https://<username>.github.io/<repo-name>/`

### Netlify
Drag the folder onto [app.netlify.com/drop](https://app.netlify.com/drop)

### Vercel
Import at [vercel.com/new](https://vercel.com/new) — framework: **Other**
